Getting Started

  1. Use Google Chrome or Firefox.

    We're working on adding support for Safari, but stick with these browsers for now.

  2. Allow your browser to access your microphone.
  3. Use headphones while recording! You'll create a feedback loop if you don't which will hurt everyone's ears.
  4. Create a new project or find an open project you would like to contribute to.
  5. Click "record" to record audio, and "stop" to save that audio to the project.

    If you aren't satisifed with you track, click "delete" to remove your track.

  6. At the bottom of the project page, invite others to collaborate on your project by specifying what instruments you want to add to the project.

    For example, if you start a project with a guitar track, you can specify that you want to add a bass track. When bass players sign in, they'll see your project as one looking for a bass part. The project owner can review the community's track contributions, and choose to keep or delete the added tracks.

  7. The project owner can set the project's status as either "open" or "complete."

    8. "Open" projects are accepting track contributions, but "closed" projects cannot be contributed to.

  8. Create, collaborate and have fun!
